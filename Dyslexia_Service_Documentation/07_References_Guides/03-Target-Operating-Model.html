children) ``` Primary Responsibilities: ├─ Monitor referral intake queue ├─ Validate incoming referrals (completeness check) ├─ Assign to screening specialist ├─ Track intake SLA (target: <24h) ├─ Escalate missing data requests └─ Maintain intake quality metrics Required Skills: ├─ Data quality assessment ├─ Customer service ├─ Process discipline ├─ Problem-solving └─ Basic data literacy Tools/Systems: ├─ ServiceNow (intake ticketing) ├─ Oracle CRM (child profile) └─ Microsoft Outlook (email + calendar) Success Metrics: ├─ Intake SLA: 100% <24h ├─ Data completeness: 100% ├─ Customer satisfaction: >85 NPS └─ Processing cost: <$5 per intake ``` #### 2. **Screening Specialist** (FTE: 1 per 1000 children) ``` Primary Responsibilities: ├─ Administer behavioral screening questionnaire ├─ Score responses + calculate risk level ├─ Apply triage rules (to assessment vs. monitoring) ├─ Recommend specialist routing (if needed) ├─ Document assessment in case record └─ Provide parent feedback on screening results Required Skills: ├─ Child development knowledge ├─ Assessment administration ├─ Communication with diverse families ├─ Clinical judgment (basic level) ├─ Attention to detail Training Required: ├─ Dyslexia awareness (16 hours) ├─ Behavioral screening methodology (8 hours) ├─ LGPD compliance (4 hours) ├─ System training (ServiceNow + Oracle) (4 hours) └─ Annual refresher (8 hours) Tools/Systems: ├─ ServiceNow (workflow + scheduling) ├─ Oracle CRM (child record) ├─ Assessment platform (screening forms) └─ Microsoft Teams (communication) Success Metrics: ├─ Screening completion rate: >95% ├─ Screening SLA: <48h ├─ Assessment accuracy (validated against specialist): >90% ├─ Parent satisfaction: >80 NPS └─ False positive rate: <10% ``` #### 3. **Assessment Specialist** (FTE: 1 per 300 children) ``` Primary Responsibilities: ├─ Administer multidimensional cognitive battery ├─ Conduct 3 assessment sessions (phonology, fluency, comprehension) ├─ Collect and validate test data ├─ Document observations + behavioral notes ├─ Prepare raw score data for automated analysis ├─ Communicate results to parents/educators └─ Escalate technical failures to support Required Skills: ├─ Dyslexia assessment expertise (master's level preferred) ├─ Child psychology ├─ Psychometric assessment knowledge ├─ Data collection + documentation ├─ Empathetic communication (especially with struggling children) Credentials: ├─ Fonoaudiólogo (speech-language pathologist) OR ├─ Psicopedagogo (educational psychologist) OR ├─ Psychologist with dyslexia specialization Training Required: ├─ Assessment battery certification (40 hours) ├─ Data quality assurance (8 hours) ├─ System training (assessment platform + Oracle) (8 hours) ├─ LGPD compliance (4 hours) └─ Annual certification renewal (16 hours) Tools/Systems: ├─ Assessment platform (test administration) ├─ Audio recording system (secure cloud storage) ├─ Oracle CRM (case documentation) ├─ ServiceNow (scheduling + escalation) └─ Power BI (progress dashboards - read-only) Success Metrics: ├─ Assessment completion rate: >98% ├─ Assessment SLA: <10 days for full battery ├─ Data quality (no missing values): 100% ├─ Test-retest reliability: >0.85 (validated) ├─ Parent satisfaction: >85 NPS └─ Assessment cost per child: <$50 ``` #### 4. **Intervention Specialist** (FTE: 1 per 150 children in active intervention) ``` Primary Responsibilities: ├─ Review AI-generated intervention plans ├─ Refine plans based on clinical judgment ├─ Approve plan (digital signature) ├─ Assign to educator implementation team ├─ Monitor plan adherence + effectiveness ├─ Escalate non-responsive cases ├─ Provide strategy consultation to educators ├─ Conduct progress reviews (monthly) Required Skills: ├─ Dyslexia intervention expertise ├─ Pedagogical knowledge ├─ AI/data interpretation ├─ Mentoring and coaching ├─ Cross-functional collaboration Credentials: ├─ Master's in Special Education OR ├─ Fonoaudiólogo with intervention experience OR ├─ Psicopedagogo with clinical track record Training Required: ├─ Intervention protocol certification (32 hours) ├─ AI-assisted planning (8 hours) ├─ Progress monitoring + data interpretation (8 hours) ├─ System training (Oracle + ServiceNow + Power BI) (8 hours) └─ LGPD compliance (4 hours) Tools/Systems: ├─ Oracle CRM (case management) ├─ ServiceNow (task orchestration) ├─ Claude API (Hermes Agent - view only) ├─ Power BI (progress dashboards) ├─ Microsoft Teams (educator communication) Success Metrics: ├─ Plan approval SLA: <4h ├─ Plan quality (educator satisfaction): >85% ├─ Intervention effectiveness (>60% show progress): >60% ├─ Non-response escalation rate: <5% ├─ Plan refinement rate: <20% (suggests AI plans are good) └─ Intervention cost per child: <$100/month ``` #### 5. **Program Manager / ONG Coordinator** (FTE: 1 per ONG + 1 state-level) ``` Primary Responsibilities: ├─ Oversee service delivery in ONG/region ├─ Manage caseload allocation (educators, specialists) ├─ Monitor SLAs + escalate breaches ├─ Coordinate with service oversight ├─ Manage educator recruitment + training ├─ Maintain compliance (LGPD, audits, reporting) ├─ Optimize operations (reduce costs, improve quality) └─ Represent ONG in governance meetings Required Skills: ├─ Program management ├─ Budget management ├─ Staff management ├─ Data analysis ├─ Stakeholder communication Credentials: ├─ MBA or equivalent management experience OR ├─ 5+ years program management in education/health Training Required: ├─ Service delivery model (16 hours) ├─ Power BI dashboards (8 hours) ├─ Compliance management (8 hours) ├─ ServiceNow administration (4 hours) └─ Annual refresher (8 hours) Tools/Systems: ├─ ServiceNow (caseload + SLA dashboard) ├─ Power BI (program metrics + KPIs) ├─ Oracle CRM (client management) ├─ Microsoft Excel (budgeting + forecasting) ├─ Jira (project management, optional) Success Metrics: ├─ SLA INTAKE (24h SLA) Unified portal intake LGPD consent digital signature ↓ 2. SCREENING (48h SLA) Behavioral questionnaire Risk classification ↓ 3. ASSESSMENT (10 days SLA) 3-session cognitive battery Multidimensional scoring ↓ 4. DIAGNOSIS (2h SLA + 48h specialist review if needed) Risk classification Specialist validation (if High Risk) ↓ 5. PLANNING (5 min AI + 2h psychopedagogue review) Personalized intervention plan Digital approval workflow ↓ 6. COMMUNICATION (1h SLA) Multi-channel notifications (SMS/Email/Push/WhatsApp) ↓ 7. INTERVENTION DELIVERY (6 months) Structured phases Regular educator sessions ↓ 8. MONITORING (weekly/bi-weekly) Progress collection Dynamic reclassification if needed ↓ 9. EXIT EVALUATION (final assessment) Impact measurement ROI calculation Recommendations for continuity ↓ Program Complete ``` --- ## 💻 TECHNOLOGY DIMENSION ### Enterprise Architecture ```
