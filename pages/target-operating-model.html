Target Operating Model (TOM)

Integrated Dyslexia Monitoring & Intervention Service Platform

TOM OVERVIEW

The Target Operating Model defines HOW the future state service will operate across four interconnected dimensions:
1. People - Roles, skills, organizational structure
2. Process - Core workflows, decision gates, SLAs
3. Technology - Systems, data, integrations
4. Governance - Policies, compliance, oversight

PEOPLE DIMENSION

Organizational Structure

service OVERSIGHT (Health/Education)

Policy definition - Compliance monitoring - Funding allocation

  • SERVICE DELIVERY NETWORK (ONG Federation) - Program delivery - Quality assurance - Stakeholder coordination
  • INTAKE CENTER | ASSESSMENT SERVICES | INTERVENTION & MONITOR

Core Roles & Responsibilities

1. Intake Coordinator (FTE: 1 per 500 children)

Primary Responsibilities:

  • Monitor referral intake queue
  • Validate incoming referrals (completeness check)
  • Assign to screening specialist
  • Track intake SLA (target: <24h)
  • Escalate missing data requests
  • Maintain intake quality metrics

Required Skills:

  • Data quality assessment
  • Customer service
  • Process discipline
  • Problem-solving
  • Basic data literacy

Tools/Systems:

  • ServiceNow (intake ticketing)
  • Oracle CRM (child profile)
  • Microsoft Outlook (email + calendar)

Success Metrics:

  • Intake SLA: 100% <24h
  • Data completeness: 100%
  • Customer satisfaction: >85 NPS
  • Processing cost: <$5 per intake

2. Screening Specialist (FTE: 1 per 1000 children)

Primary Responsibilities:

  • Administer behavioral screening questionnaire
  • Score responses + calculate risk level
  • Apply triage rules (to assessment vs. monitoring)
  • Recommend specialist routing (if needed)
  • Document assessment in case record
  • Provide parent feedback on screening results

Required Skills:

  • Child development knowledge
  • Assessment administration
  • Communication with diverse families
  • Clinical judgment (basic level)
  • Attention to detail

Training Required:

  • Dyslexia awareness (16 hours)
  • Behavioral screening methodology (8 hours)
  • LGPD compliance (4 hours)
  • System training (ServiceNow + Oracle) (4 hours)
  • Annual refresher (8 hours)

Tools/Systems:

  • ServiceNow (workflow + scheduling)
  • Oracle CRM (child record)
  • Assessment platform (screening forms)
  • Microsoft Teams (communication)

Success Metrics:

  • Screening completion rate: >95%
  • Screening SLA: <48h
  • Assessment accuracy (validated against specialist): >90%
  • Parent satisfaction: >80 NPS
  • False positive rate: <10%

3. Assessment Specialist (FTE: 1 per 300 children)

Primary Responsibilities:

  • Administer multidimensional cognitive battery
  • Conduct 3 assessment sessions (phonology, fluency, comprehension)
  • Collect and validate test data
  • Document observations + behavioral notes
  • Prepare raw score data for automated analysis
  • Communicate results to parents/educators
  • Escalate technical failures to support

Required Skills:

  • Dyslexia assessment expertise (master's level preferred)
  • Child psychology
  • Psychometric assessment knowledge
  • Data collection + documentation
  • Empathetic communication (especially with struggling children)

Credentials:

  • Speech Therapist (speech-language pathologist) OR
  • Psychopedagogue (educational psychologist) OR
  • Psychologist with dyslexia specialization

Training Required:

  • Assessment battery certification (40 hours)
  • Data quality assurance (8 hours)
  • System training (assessment platform + Oracle) (8 hours)
  • LGPD compliance (4 hours)
  • Annual certification renewal (16 hours)

Tools/Systems:

  • Assessment platform (test administration)
  • Audio recording system (secure cloud storage)
  • Oracle CRM (case documentation)
  • ServiceNow (scheduling + escalation)
  • Power BI (progress dashboards - read-only)

Success Metrics:

  • Assessment completion rate: >98%
  • Assessment SLA: <10 days for full battery
  • Data quality (no missing values): 100%
  • Test-retest reliability: >0.85 (validated)
  • Parent satisfaction: >85 NPS
  • Assessment cost per child: <$50

4. Intervention Specialist (FTE: 1 per 150 children in active intervention)

Primary Responsibilities:

  • Review AI-generated intervention plans
  • Refine plans based on clinical judgment
  • Approve plan (digital signature)
  • Assign to educator implementation team
  • Monitor plan adherence + effectiveness
  • Escalate non-responsive cases
  • Provide strategy consultation to educators
  • Conduct progress reviews (monthly)

Required Skills:

  • Dyslexia intervention expertise
  • Pedagogical knowledge
  • AI/data interpretation
  • Mentoring and coaching
  • Cross-functional collaboration

Credentials:

  • Master's in Special Education OR
  • Speech Therapist with intervention experience OR
  • Psychopedagogue with clinical track record

Training Required:

  • Intervention protocol certification (32 hours)
  • AI-assisted planning (8 hours)
  • Progress monitoring + data interpretation (8 hours)
  • System training (Oracle + ServiceNow + Power BI) (8 hours)
  • LGPD compliance (4 hours)

Tools/Systems:

  • Oracle CRM (case management)
  • ServiceNow (task orchestration)
  • Claude API (Hermes Agent - view only)
  • Power BI (progress dashboards)
  • Microsoft Teams (educator communication)

Success Metrics:

  • Plan approval SLA: <4h
  • Plan quality (educator satisfaction): >85%
  • Intervention effectiveness (>60% show progress): >60%
  • Non-response escalation rate: <5%
  • Plan refinement rate: <20% (suggests AI plans are good)
  • Intervention cost per child: <$100/month

5. Program Manager / ONG Coordinator (FTE: 1 per ONG + 1 state-level)

Primary Responsibilities:

  • Oversee service delivery in ONG/region
  • Manage caseload allocation (educators, specialists)
  • Monitor SLAs + escalate breaches
  • Coordinate with service oversight
  • Manage educator recruitment + training
  • Maintain compliance (LGPD, audits, reporting)
  • Optimize operations (reduce costs, improve quality)
  • Represent ONG in governance meetings

Required Skills:

  • Program management
  • Budget management
  • Staff management
  • Data analysis
  • Stakeholder communication

Credentials:

  • MBA or equivalent management experience OR
  • 5+ years program management in education/health

Training Required:

  • Service delivery model (16 hours)
  • Power BI dashboards (8 hours)
  • Compliance management (8 hours)
  • ServiceNow administration (4 hours)
  • Annual refresher (8 hours)

Tools/Systems:

  • ServiceNow (caseload + SLA dashboard)
  • Power BI (program metrics + KPIs)
  • Oracle CRM (client management)
  • Microsoft Excel (budgeting + forecasting)
  • Jira (project management, optional)

Success Metrics:

  • SLA compliance: >95% across all services
  • Budget adherence: ±5% of plan
  • Educator retention: >85%
  • Program efficiency: <$150 cost per child/month
  • service compliance: 100%
  • Stakeholder satisfaction: >80 NPS

6. Data Governance & Compliance Lead (FTE: 1 per service entity)

Primary Responsibilities:

  • Ensure LGPD compliance across all ONGs
  • Audit data handling + access controls
  • Manage consent records + deletion requests
  • Monitor audit trails (proof of compliance)
  • Review data privacy incidents
  • Report to service regulators
  • Train teams on LGPD requirements
  • Maintain data governance policies

Required Skills:

  • Data privacy expertise (LGPD/GDPR)
  • Audit + compliance
  • Risk management
  • Regulatory knowledge
  • Data security fundamentals

Credentials:

  • Privacy Certification (CDPO, CIPM, or equivalent) OR
  • 5+ years legal/compliance background

Training Required:

  • LGPD deep-dive (24 hours)
  • Data governance frameworks (8 hours)
  • System auditing (Oracle, ServiceNow) (8 hours)
  • Incident response (8 hours)
  • Annual refresher (12 hours)

Tools/Systems:

  • ServiceNow (audit workflows)
  • Oracle Data Governance (metadata + lineage)
  • Microsoft Power BI (compliance dashboards)
  • Azure Information Protection (data classification)
  • Compliance software (OneTrust, Drata, or similar)

Success Metrics:

  • LGPD compliance: 100%
  • Audit findings: 0 critical/high severity
  • Consent record completeness: 100%
  • Data deletion request SLA: 100% <30 days
  • Audit trail availability: 100% of transactions
  • Regulatory audit results: Pass with no findings

7. System Administrator & Support (FTE: 1 per 2000 children system-wide)

Primary Responsibilities:

  • User account management (access provisioning)
  • System monitoring (uptime, performance)
  • Password resets + account recovery
  • Help desk support (Tier 1)
  • System backups + disaster recovery
  • Apply security patches + updates
  • Document system configurations
  • Train users on system features

Required Skills:

  • IT operations
  • Cloud infrastructure
  • Help desk support
  • Basic cybersecurity
  • Documentation + training

Certifications:

  • CompTIA A+ OR
  • Azure Fundamentals (AZ-900) OR
  • ServiceNow Administrator (optional)

Training Required:

  • Oracle CRM administration (24 hours)
  • ServiceNow administration (24 hours)
  • Azure fundamentals (16 hours)
  • Security best practices (8 hours)
  • Annual refresher (8 hours)

Tools/Systems:

  • Oracle Cloud (system administration)
  • ServiceNow (workflow administration)
  • Microsoft Azure (cloud management)
  • Jira (incident tracking)
  • VPN + Access controls (Azure AD)

Success Metrics:

  • System uptime: >99.5%
  • Help desk response SLA: <2 hours
  • Password reset SLA: <1 hour
  • Security patch deployment: <30 days
  • User onboarding SLA: <5 days
  • Support cost per user: <$20/year

Process Flow (Simplified)

Child Referred
↓ 1. INTAKE (24h SLA) - Unified portal intake - LGPD consent digital signature
↓ 2. SCREENING (48h SLA) - Behavioral questionnaire - Risk classification
↓ 3. ASSESSMENT (10 days SLA) - 3-session cognitive battery - Multidimensional scoring
↓ 4. DIAGNOSIS (2h SLA + 48h specialist review if needed) - Risk classification - Specialist validation (if High Risk)
↓ 5. PLANNING (5 min AI + 2h psychopedagogue review) - Personalized intervention plan - Digital approval workflow
↓ 6. COMMUNICATION (1h SLA) - Multi-channel notifications (SMS/Email/Push/WhatsApp)
↓ 7. INTERVENTION DELIVERY (6 months) - Structured phases - Regular educator sessions
↓ 8. MONITORING (weekly/bi-weekly) - Progress collection - Dynamic reclassification if needed
↓ 9. EXIT EVALUATION (final assessment) - Impact measurement - ROI calculation - Recommendations for continuity
↓ Program Complete

  • Health systems (API for specialist referrals)
  • Payment systems (billing + grant tracking)

  • Analytics platforms (external research partnerships)

Technology Stack Details

LAYER 1: CITIZEN INTERFACES

  • Web Portal: Angular 16 (responsive, accessible)
  • Mobile Apps: React Native (iOS + Android)
  • WhatsApp Integration: Twilio WhatsApp API
  • Email: SendGrid (templated, tracked)

LAYER 2: API & ORCHESTRATION

  • API Gateway: Azure API Management
  • Workflow Orchestration: ServiceNow Workflow Engine
  • Authentication: Azure AD (OAuth 2.0, MFA)
  • Rate Limiting & Monitoring: API Management
  • Message Queue: Azure Service Bus (async processing)

LAYER 3: CORE BUSINESS SYSTEMS

  • CRM: Oracle Service Cloud (child master profile, case mgmt)
  • Workflow/Ticketing: ServiceNow (intake, screening, scheduling)
  • Power Platform: Power Apps (intake forms), Power Automate (workflows), Power BI (dashboards)
  • Assessment Engine: Custom platform (testing + AI scoring)
  • AI/Reasoning: Claude API (Hermes Agent for intervention planning)

LAYER 4: DATA & ANALYTICS

  • Data Lake: Azure Data Lake Storage (Gen 2)
  • Data Warehouse: Synapse Analytics (SQL + Spark)
  • Analytics: Power BI (120+ dashboards across roles)
  • Data Governance: Oracle Data Governance (metadata, lineage, quality)
  • Machine Learning: Azure ML (predictive models for reclassification)

LAYER 5: SECURITY & COMPLIANCE

  • Encryption: AES-256 (at rest), TLS 1.3 (in transit)
  • Access Control: Azure AD RBAC + Oracle fine-grained access
  • Audit Logging: All system activities logged to Azure Monitor
  • Data Privacy: Azure Information Protection (PII classification)
  • Backup & Recovery: Geo-redundant backups (24h RPO, 4h RTO)
  • Compliance Monitoring: Automated LGPD compliance checks
