2. Process - Core workflows, decision gates, SLAs

Core Roles & Responsibilities

1. Intake Coordinator (FTE: 1 per 500 children)

Primary Responsibilities:
• Monitor referral intake queue
• Validate incoming referrals (completeness check)
• Assign to screening specialist
• Track intake SLA (target: <24h)
• Escalate missing data requests
• Maintain intake quality metrics Required Skills:
• Data quality assessment
• Customer service
• Process discipline
• Problem-solving
• Basic data literacy Tools/Systems:
• ServiceNow (intake ticketing)
• Oracle CRM (child profile)
• Microsoft Outlook (email + calendar) Success Metrics:
• Intake SLA: 100% <24h
• Data completeness: 100%
• Customer satisfaction: >85 NPS
• Processing cost: <$5 per intake

2. Screening Specialist (FTE: 1 per 1000 children)

Primary Responsibilities:
• Administer behavioral screening questionnaire
• Score responses + calculate risk level
• Apply triage rules (to assessment vs. monitoring)
• Recommend specialist routing (if needed)
• Document assessment in case record
• Provide parent feedback on screening results Required Skills:
• Child development knowledge
• Assessment administration
• Communication with diverse families
• Clinical judgment (basic level)
• Attention to detail Training Required:
• Dyslexia awareness (16 hours)
• Behavioral screening methodology (8 hours)
• LGPD compliance (4 hours)
• System training (ServiceNow + Oracle) (4 hours)
• Annual refresher (8 hours) Tools/Systems:
• ServiceNow (workflow + scheduling)
• Oracle CRM (child record)
• Assessment platform (screening forms)
• Microsoft Teams (communication) Success Metrics:
• Screening completion rate: >95%
• Screening SLA: <48h
• Assessment accuracy (validated against specialist): >90%
• Parent satisfaction: >80 NPS
• False positive rate: <10%

3. Assessment Specialist (FTE: 1 per 300 children)

Primary Responsibilities:
• Administer multidimensional cognitive battery
• Conduct 3 assessment sessions (phonology, fluency, comprehension)
• Collect and validate test data
• Document observations + behavioral notes
• Prepare raw score data for automated analysis
• Communicate results to parents/educators
• Escalate technical failures to support Required Skills:
• Dyslexia assessment expertise (master's level preferred)
• Child psychology
• Psychometric assessment knowledge
• Data collection + documentation
• Empathetic communication (especially with struggling children) Credentials:
• Fonoaudiólogo (speech-language pathologist) OR
• Psicopedagogo (educational psychologist) OR
• Psychologist with dyslexia specialization Training Required:
• Assessment battery certification (40 hours)
• Data quality assurance (8 hours)
• System training (assessment platform + Oracle) (8 hours)
• LGPD compliance (4 hours)
• Annual certification renewal (16 hours) Tools/Systems:
• Assessment platform (test administration)
• Audio recording system (secure cloud storage)
• Oracle CRM (case documentation)
• ServiceNow (scheduling + escalation)
• Power BI (progress dashboards - read-only) Success Metrics:
• Assessment completion rate: >98%
• Assessment SLA: <10 days for full battery
• Data quality (no missing values): 100%
• Test-retest reliability: >0.85 (validated)
• Parent satisfaction: >85 NPS
• Assessment cost per child: <$50

4. Intervention Specialist (FTE: 1 per 150 children in active intervention)

Primary Responsibilities:
• Review AI-generated intervention plans
• Refine plans based on clinical judgment
• Approve plan (digital signature)
• Assign to educator implementation team
• Monitor plan adherence + effectiveness
• Escalate non-responsive cases
• Provide strategy consultation to educators
• Conduct progress reviews (monthly) Required Skills:
• Dyslexia intervention expertise
• Pedagogical knowledge
• AI/data interpretation
• Mentoring and coaching
• Cross-functional collaboration Credentials:
• Master's in Special Education OR
• Fonoaudiólogo with intervention experience OR
• Psicopedagogo with clinical track record Training Required:
• Intervention protocol certification (32 hours)
• AI-assisted planning (8 hours)
• Progress monitoring + data interpretation (8 hours)
• System training (Oracle + ServiceNow + Power BI) (8 hours)
• LGPD compliance (4 hours) Tools/Systems:
• Oracle CRM (case management)
• ServiceNow (task orchestration)
• Claude API (Hermes Agent - view only)
• Power BI (progress dashboards)
• Microsoft Teams (educator communication) Success Metrics:

• Non-response escalation rate: <5%
• Plan refinement rate: <20% (suggests AI plans are good)
• Intervention cost per child: <$100/month

Technology Stack Details

LAYER 1: CITIZEN INTERFACES
• Web Portal: Angular 16 (responsive, accessible)
• Mobile Apps: React Native (iOS + Android)
• WhatsApp Integration: Twilio WhatsApp API
• Email: SendGrid (templated, tracked) LAYER 2: API & ORCHESTRATION
• API Gateway: Azure API Management
• Workflow Orchestration: ServiceNow Workflow Engine
• Authentication: Azure AD (OAuth 2.0, MFA)
• Rate Limiting & Monitoring: API Management
• Message Queue: Azure Service Bus (async processing) LAYER 3: CORE BUSINESS SYSTEMS
• CRM: Oracle Service Cloud (child master profile, case mgmt)
• Workflow/Ticketing: ServiceNow (intake, screening, scheduling)
• Power Platform: Power Apps (intake forms), Power Automate (workflows), Power BI (dashboards)
• Assessment Engine: Custom platform (testing + AI scoring)
• AI/Reasoning: Claude API (Hermes Agent for intervention planning) LAYER 4: DATA & ANALYTICS
• Data Lake: Azure Data Lake Storage (Gen 2)
• Data Warehouse: Synapse Analytics (SQL + Spark)
• Analytics: Power BI (120+ dashboards across roles)
• Data Governance: Oracle Data Governance (metadata, lineage, quality)
• Machine Learning: Azure ML (predictive models for reclassification) LAYER 5: SECURITY & COMPLIANCE
• Encryption: AES-256 (at rest), TLS 1.3 (in transit)
• Access Control: Azure AD RBAC + Oracle fine-grained access
• Audit Logging: All system activities logged to Azure Monitor
• Data Privacy: Azure Information Protection (PII classification)
• Backup & Recovery: Geo-redundant backups (24h RPO, 4h RTO)
• Compliance Monitoring: Automated LGPD compliance checks
