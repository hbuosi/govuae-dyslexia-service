Current State vs Future State Analysis

Dyslexia Monitoring & Intervention Service Transformation

📊 CURRENT STATE (AS-IS) PAIN POINTS

1. Fragmented Child Assessment & Intake

Problem:

🔴 Multiple entry points with no unified intake Educators refer via email/phone (no formal process) Parents register through different ONG systems Specialists have separate referral channels No centralized child record repository Duplicate registrations common (same child in 3+ systems)

Impact:
- ⚠️ Operational Waste: 20-30% time lost to data reconciliation
- ⚠️ Service Quality: Child starts assessment multiple times
- ⚠️ Data Integrity: Inconsistent demographic data across systems
- ⚠️ Compliance Risk: LGPD consent fragmented across systems

2. Manual Referral Delays & Bottlenecks

Problem:

🔴 Referral-to-Assessment timeline is unpredictable Educator refers → Email sits in inbox (2-5 days) Manual screening by caseworker (3-7 days) Assessment scheduling via phone/email (3-10 days) No automated escalation triggers Current SLA: 21-35 days (should be 7-10 days)

Impact:
- ⚠️ Service Delay: Average 28-day gap between referral and assessment start
- ⚠️ Child Opportunity Cost: Early intervention window closes
- ⚠️ Educator Frustration: No visibility into referral status
- ⚠️ System Inefficiency: Caseworkers spend 40% time on scheduling admin

3. Inconsistent Specialist Triage & Validation

Problem:

🔴 No standard triage rules for specialist referrals Different ONG coordinators use different criteria Specialist selection is ad-hoc (based on personal relationships) No SLA for specialist review (ranges 1-14 days) Lack of escalation protocol for critical cases No feedback loop from specialists to ONG

Impact:
- ⚠️ Service Inconsistency: Same child gets different pathways depending on ONG
- ⚠️ Equity Issue: Access to specialists varies by geography/wealth
- ⚠️ Quality Variance: Some cases get expert review, others don't
- ⚠️ No Learning Loop: System doesn't improve based on specialist feedback

4. Lack of Cross-Stakeholder Visibility

Problem:

🔴 Stakeholders operate in information silos Parents don't know child's status (no regular updates) Educators don't see progress tracking (no data access) Specialists don't know if recommendations were implemented ONG coordinators don't have holistic caseload view No shared dashboard or notification system Communication happens via email (untracked, scattered)

Impact:
- ⚠️ Parent Anxiety: Lack of communication breeds distrust
- ⚠️ Educator Disconnect: Teachers don't see ROI of referral
- ⚠️ Specialist Frustration: Can't track if advice was followed
- ⚠️ Program Ineffectiveness: No feedback loop = no improvement

5. No Standard Service Taxonomy or Classification

Problem:

🔴 Service definition varies by ONG Each ONG defines "dislexia" differently Risk classification thresholds differ Intervention phases are non-standard "Progress" is measured differently across ONGs No shared vocabulary (educators and specialists use different terms) Billing/reporting cannot be aggregated

Impact:
- ⚠️ Data Aggregation Impossible: Can't compare across ONGs
- ⚠️ Policy Blind Spot: service can't measure service maturity
- ⚠️ Quality Metrics Weak: KPIs are anecdotal, not systematic
- ⚠️ Scaling Difficulty: Every new ONG requires custom integration

6. Limited KPI Governance & Reporting

Problem:

🔴 Metrics are ad-hoc and untracked No centralized KPI dashboard Reporting is manual (Excel spreadsheets) Historical data not retained (hard to see trends) SLAs exist but aren't monitored (no enforcement) No audit trail of decision-making Monthly reports take 20% of coordinator time

Impact:
- ⚠️ Accountability Gap: No way to prove service effectiveness
- ⚠️ Decision Blindness: Leaders can't see what's working/failing
- ⚠️ Audit Risk: Can't demonstrate LGPD compliance on demand
- ⚠️ Funding Risk: Can't justify program effectiveness to donors

7. Disconnected Technology Ecosystem

Problem:

🔴 Systems don't talk to each other Assessment tool: standalone desktop app Referral tracking: email + Google Sheets Parent communication: WhatsApp (untracked) Progress data: manual entry in local database Reporting: manual export to Excel No API integrations, no data flow automation LGPD compliance is manual (consent forms in PDFs)

Impact:
- ⚠️ Data Silos: Same data re-entered in multiple systems
- ⚠️ Manual Errors: 5-10% data entry mistakes
- ⚠️ Compliance Risk: Can't prove audit trail
- ⚠️ Scalability Block: Can't scale without major IT overhaul

🎯 FUTURE STATE (TO-BE) VISION

1. Unified, Citizen-Centric Intake Workflow

Solution:

🟢 Single digital intake portal (web + mobile) Educator referral form (integrated with school system) Parent self-registration (with digital LGPD consent) Specialist direct referral (via OAuth integration) Automatic duplicate detection (by CPF/name matching) Single child master profile (golden record) Real-time intake status visibility (for all stakeholders)

Benefits:
- ✅ Speed: Intake → Assessment in 7 days (vs. 28 days current)
- ✅ Quality: No duplicate registrations
- ✅ Compliance: Digitally signed LGPD consent with audit trail
- ✅ UX: Citizens see clear progress (reduces anxiety)

Technology:
- Oracle Service Cloud (citizen portal)
- ServiceNow (intake orchestration)
- Azure AD (authentication)

2. Standardized, Rules-Based Triage & Assessment

Solution:

🟢 Automated triage workflow with clear decision gates Behavioral screening (standardized questionnaire) Risk classification rules (consistent across ONGs) Automated specialist routing (by geography + expertise) SLA tracking (with escalation if >48h) Auto-assignment to assessment specialist Real-time status updates (SMS/email to stakeholders)

Benefits:
- ✅ Speed: Triage decision in <2 hours
- ✅ Equity: Same criteria apply everywhere (no geography bias)
- ✅ Quality: Every high-risk case reviewed (no gaps)
- ✅ Transparency: Educators see exactly why child got routed to specialist

Technology:
- ServiceNow Workflow Engine (triage logic)
- Microsoft Power Automate (automated routing)
- Oracle Case Management (case tracking)

3. Integrated Assessment & Diagnosis

Solution:

🟢 Automated cognitive battery + AI analysis Standardized BPMN workflow (same for all children) Multidimensional assessment (phonology, fluency, comprehension) Real-time scoring + risk classification Hermes Agent AI generates personalized plan Specialist review gate (for high-risk cases) Digital signature workflow (plan approval)

Benefits:
- ✅ Consistency: Same assessment used everywhere (benchmarkable)
- ✅ Speed: Diagnosis in 10 days (vs. 21-35 days current)
- ✅ Quality: AI-assisted planning (no human bias)
- ✅ Compliance: Full audit trail of all decisions

Technology:
- Oracle WPM test platform (audio assessment)
- Claude API (Hermes Agent - intervention planning)
- ServiceNow (approval workflows)

4. Cross-Stakeholder Visibility Dashboard

Solution:

🟢 Microsoft Power BI dashboards (role-based) Parent View: Child progress, next steps, resources Educator View: Child performance, recommended strategies, class impact Specialist View: Caseload, outcomes, intervention effectiveness ONG Coordinator View: Program metrics, resource allocation, SLAs service View: Aggregate metrics, equity analysis, program maturity Real-time notifications (SMS/email/push) for key events

Benefits:
- ✅ Engagement: Parents/educators see value (stay in program)
- ✅ Accountability: Everyone sees agreed SLAs + actual performance
- ✅ Learning: Data reveals what's working (feedback loop)
- ✅ Scaling: service can measure maturity across ONGs

Technology:
- Microsoft Power BI (data visualization)
- Azure Data Lake (centralized data repository)
- Twilio (multi-channel notifications)

5. Standardized Service Taxonomy & Governance

Solution:

🟢 Shared service model with governance checkpoints Standard child status taxonomy (Registered → Screened → Assessed → In Progress → Exited) Uniform risk classification (5 tiers: No Risk, Low, Moderate, High, Critical) Standardized intervention phases (same naming/sequencing everywhere) Common KPI definitions (same child counted same way across ONGs) Shared glossary (educators + specialists use same vocabulary) Service level agreements (published, monitored, enforced)

Benefits:
- ✅ Comparability: Can aggregate data across ONGs (10x reporting efficiency)
- ✅ Policy Visibility: service sees actual service maturity
- ✅ Scaling: New ONG adopts existing model (vs. custom build)
- ✅ Quality: Standards create floor (prevents low-quality variations)

Technology:
- Oracle Data Governance (metadata management)
- ServiceNow (policy/SLA management)
- SharePoint (shared documentation)

6. Automated KPI Governance & Real-Time Reporting

Solution:

🟢 Centralized performance management system Real-time KPI dashboard (updated hourly) SLA tracking with automated alerts (if >threshold) Historical trend analysis (identify improvements) Audit trail logging (every decision recorded) Auto-generated monthly reports (no manual work) Governance board review (data-driven decisions) Citizen-facing service quality dashboard (transparency)

Benefits:
- ✅ Accountability: Real-time visibility to SLA performance
- ✅ Rapid Response: Alerts trigger when SLAs at risk
- ✅ Data-Driven: Leaders see trends + can make informed decisions
- ✅ Compliance: Audit trail proves LGPD compliance on demand
- ✅ Efficiency: Reports auto-generated (saves 40h/month per ONG)

Technology:
- Microsoft Power BI (KPI visualization)
- Azure Synapse (data analytics)
- ServiceNow (workflow + alert management)

7. Integrated, API-First Technology Ecosystem

Solution:

🟢 Enterprise platform integrations Oracle CRM: Single citizen/child master profile ServiceNow: Unified workflow orchestration + ticketing Microsoft Power Platform: Dashboards, automations, mobile apps Assessment Tool API: Real-time scoring + data sync Notification Service: Multi-channel (SMS/Email/Push) Data Lake: Centralized analytics + LGPD compliance Single sign-on (Azure AD) for all users

Benefits:
- ✅ Efficiency: No more manual data entry (API automation)
- ✅ Accuracy: Single source of truth (no duplicate data)
- ✅ Scalability: Same platform supports any ONG
- ✅ Compliance: Data governance built-in (LGPD, audit trail)
- ✅ Analytics: Centralized data enables AI/ML improvements

Technology:
- Oracle Cloud (CRM + data management)
- ServiceNow Cloud (workflow + ticketing)
- Microsoft Azure (analytics + AI)
